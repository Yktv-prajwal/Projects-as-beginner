# 🔢 Even and Odd Number Checker in Python

## 📌 Project Overview
This Python program checks whether a user-entered number is even or odd. The user inputs an integer, and the program uses the modulus operator (`%`) along with conditional statements to determine the result.

## 🚀 Features
- Accepts an integer as user input.
- Determines whether the number is even or odd.
- Displays the appropriate result.
- Simple and beginner-friendly project.
- Demonstrates the use of conditional statements and arithmetic operators.

## 📚 Concepts Used
- User Input (`input()`)
- Type Casting (`int()`)
- Variables
- Modulus Operator (`%`)
- Conditional Statements (`if-else`)
- Output using `print()`

## 🔍 Working of the Program

### Step 1: Take User Input
The program asks the user to enter an integer.

### Step 2: Check the Number
The modulus operator (`%`) is used to check whether the number is divisible by 2.

- If the remainder is `0`, the number is even.
- Otherwise, the number is odd.

### Step 3: Display the Result
The program prints whether the entered number is even or odd.

## 💻 Sample Output

```
8 is EVEN Number
```

```
Enter Your Number: 7
7 is ODD Number
```
